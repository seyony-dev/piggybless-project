 Piggy Blessing Bot 🐖✨

Telegram-бот, который отправляет ежедневные утренние благословения от милой свинки и позволяет "погладить" её в любое время для получения заряда позитива. Написан на Python с использованием асинхронной библиотеки `aiogram`.

## Особенности проекта
- **Ежедневные благословения**: Бот автоматически рассылает случайные добрые сообщения и стикеры со свинкой каждый день случайным образом между 08:00 и 11:00 утра каждому зарегистрированному пользователю.
- **Интерактивность**: Доступна кнопка **«Погладить свинку»**, в ответ на которую бот присылает специальный реакционный стикер.
- **Асинхронность**: Проект построен с использованием современных асинхронных библиотек (`aiogram` v3 и `apscheduler`).
- **База данных**: Легковесная `aiosqlite` для хранения ID пользователей и чатов, подписанных на ежедневную рассылку.

## Технологии
- Python 3.9+
- [aiogram](https://docs.aiogram.dev/) — фреймворк для Telegram Bot API
- [APScheduler](https://apscheduler.readthedocs.io/) — планировщик задач
- `aiosqlite` — асинхронная работа с SQLite базой данных (`users.db`)
- `python-dotenv` — загрузка переменных окружения

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/seyony-dev/piggybless-project.git
   cd piggybless-project
   ```

2. **Создайте виртуальное окружение и установите зависимости:**
   ```bash
   python -m venv venv
   # Активация окружения (Windows):
   venv\Scripts\activate
   # Активация окружения (Linux/Mac):
   source venv/bin/activate

   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   Создайте файл конфигурации `.env` в корневой директории проекта и добавьте токен вашего Telegram-бота (полученный в [@BotFather](https://t.me/BotFather)):
   ```env
   BOT_TOKEN=ваш_токен_бота_от_BotFather
   ```

4. **Запустите бота:**
   Вы можете использовать подготовленный скрипт:
   ```cmd
   start.bat
   ```
   *Или запустить вручную через Python:*
   ```bash
   python bot.py
   ```

## Структура проекта
- `bot.py` — главный файл для запуска бота и инициализации базы данных/планировщика.
- `config.py` — загрузка токена и конфигурации из переменных окружения.
- `db.py` — функции для работы с SQLite (добавление пользователей, получение списка).
- `handlers.py` — обработчики команд пользователя (`/start` и кнопка "Погладить свинку").
- `scheduler.py` — логика инициализации APScheduler и ежедневной рассылки сообщений со стикерами.
- `stickers.py` — файл, содержащий ID необходимых стикеров Telegram.
- `stickers/` — директория для хранения исходных файлов стикеров.
